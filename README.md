# ERP & CRM Operations Portal

## Overview

This project is a full-stack ERP and CRM web application developed using React, TypeScript, Node.js, and Express. It provides modules for customer management, product inventory, stock tracking, and sales challan generation.

## Features

- Secure user login
- Customer management
- Product management
- Inventory tracking
- Sales challan generation
- Dashboard

## Technologies Used

- React
- TypeScript
- Node.js
- Express
- JSON-based local storage

## Installation

```bash
npm install
npm run dev
```

Open:

http://localhost:3000
