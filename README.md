# 🚀 Mini ERP & CRM Portal

A modern **Enterprise Resource Planning (ERP)** and **Customer Relationship Management (CRM)** web application built using **React, TypeScript, Express.js, and Vite**. The application helps businesses efficiently manage customers, inventory, quotations, challans, and overall business operations through a clean and responsive interface.

# 📌 Features

### 👥 Customer Management
- Add new customers
- Edit customer information
- Delete customer records
- Search customers quickly

### 📦 Inventory Management
- Add products
- Update product details
- Track available stock
- Delete products

### 📄 Quotation Management
- Create quotations
- Manage quotation records
- Generate professional quotations

### 🚚 Challan Management
- Create delivery challans
- View challan history
- Manage dispatch records

### 📊 Dashboard
- Business overview
- Quick navigation
- Modern responsive UI

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Mahith21/mini-erp-crm.git
```

Move into the project directory

```bash
cd mini-erp-crm
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm run dev
```

The application will start on

```
http://localhost:3000
```

---

# 🚀 Production Build

Build the project

```bash
npm run build
```

Start the production server

```bash
npm start
```
